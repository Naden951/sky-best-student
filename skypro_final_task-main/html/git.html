Базовые команды терминала

  • git init - создает новый локальный Git-репозиторий в текущей папке проекта.
  • git clone - копирует существующий удаленный репозиторий на компьютер.
  • git status - показывает состояние файлов в рабочем каталоге (какие файлы изменены, добавлены или подготовлены к коммиту).
  • git add - добавляет измененные или новые файлы в область подготовленных файлов для следующего коммита.
  • git commit -m "сообщение" - сохраняет подготовленные изменения в истории репозитория с кратким описанием.
  • git push - отправляет все сделанные локальные коммиты в удаленный репозиторий.
  • git pull - скачивает последние изменения из удаленного репозитория и автоматически объединяет их с кодом.
